# Support

## How to Get Help

This project uses [GitHub Issues](https://github.com/sethbacon/azure-pipelines-terraform/issues) to track bugs and feature requests. Before filing a new issue, please search existing issues to avoid duplicates.

For questions about how to use the extension or configure specific scenarios, open a [GitHub Discussion](https://github.com/sethbacon/azure-pipelines-terraform/discussions) if available, or a GitHub Issue labelled `question`.

## Scope

Support is limited to the resources listed above. This is a community project and is not backed by any commercial support agreement.
